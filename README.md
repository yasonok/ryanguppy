# Aquarium Studio - 專業孔雀魚專賣店

🐠 專業孔雀魚養殖與販售網站

## 功能特色

- 📱 **手機後台管理** - 用手機就能新增、編輯、刪除商品
- ⚡ **即時同步** - 商品更新後顧客立即看到
- 📸 **圖片上傳** - 支援imgur等圖床連結
- 💬 **LINE 客服整合** - 一鍵聯繫購買

## 開始使用

### 1. 設定 Supabase

1. 建立 [Supabase](https://supabase.com) 帳號（免費）
2. 建立新專案
3. 建立 `products` 資料表：

```sql
CREATE TABLE products (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  type TEXT,
  gender TEXT,
  price INTEGER NOT NULL,
  stock INTEGER DEFAULT 0,
  image_url TEXT,
  status TEXT DEFAULT 'available',
  note TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

4. 設定 Storage（用於圖片）

### 2. 修改設定

編輯 `js/app.js` 和 `js/admin.js`，填入您的 Supabase 連接資訊：

```javascript
const SUPABASE_URL = 'YOUR_SUPABASE_URL';
const SUPABASE_ANON_KEY = 'YOUR_SUPABASE_ANON_KEY';
```

### 3. 部署到 Vercel

```bash
# 方法一：GitHub 匯入
1. 將此 repo 推送到 GitHub
2. 登入 Vercel.com
3. Import GitHub repository
4. Deploy！

# 方法二：Vercel CLI
npm i -g vercel
vercel
```

## 使用說明

### 新增商品

1. 打開 `/admin.html`
2. 點擊「➕ 新增商品」
3. 填入名稱、價格、庫存
4. 輸入圖片網址（如 imgur 連結）
5. 選擇狀態（上架中/已售出/保留）
6. 儲存

### 手機上傳

直接在手機瀏覽器打開 `/admin.html`，登入後即可管理商品！
